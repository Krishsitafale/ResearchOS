# Numerical Gradient

## Definition

The numerical gradient estimates how much the loss changes when a parameter changes by a very small amount.

---

## Formula

gradient ≈ (L(w+ε)-L(w))/ε

---

## Advantages

Easy to understand

Easy to implement

Useful for debugging

---

## Disadvantages

Slow

Approximation only

Not suitable for large neural networks

---

## Applications

Gradient Checking

Optimization

Machine Learning

Deep Learning

---

## Connection to ResearchOS

Understanding numerical gradients prepares us to understand analytical gradients, which are used to train embedding models and neural networks.
